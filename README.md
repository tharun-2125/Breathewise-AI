# 🌫️ BreatheWise AI

AI-powered safe route planning system that selects the safest travel route based on:

- AQI (Air Quality Index)
- Travel duration
- Passenger health profiles
- Pollution exposure calculation

## Features

- Multi-route comparison
- AQI-based risk detection
- Health-aware travel planning
- Interactive map visualization

## Tech Stack

- Python
- Streamlit
- Folium
- OpenWeather API
- OpenRouteService API
